# Angular Stomp

Simple angular stomp library.

## Using

Install via npm
```
npm i @elderbyte/ngx-stomp --save
```

Import via JS import statement

```
import {Stomp} from "@elderbyte/ngx-stomp";
```

## Updating 

* Make changes
* Update version accordingly (in package.json and dist/package.json
* `cd dist/`
* `npm publish`
